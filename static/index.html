Docker

Docker — программное обеспечение для автоматизации развёртывания и управления приложениями в средах c поддержкой контейнеризации, контейнеризатор приложений.

Основная информация:

  • Тип: свободное и открытое программное обеспечение.
  • Автор: Соломон Хайкс.
  • Написана на: Go.
  • Операционные системы: Linux, Windows, macOS и Unix-подобная операционная система.
